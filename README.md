# Bakes So Royal — Studio App

My invoice & order tracker. The whole app is one file: index.html

## Live link
https://yourname.github.io/bakes/

## How to update
1. Replace index.html in this repo with the new version (Add file → Upload files, same name).
2. Commit changes.
3. The home-screen app updates within a minute. Same link = orders kept.

## Important
- Invoices/menu/settings are saved on my phone, tied to the link above.
- Keep the link the same or saved data resets.
- Back up important invoices as PDFs (share button in the app).
